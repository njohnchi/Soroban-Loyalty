# Public Audit Reports

Place final third-party smart-contract audit reports and remediation summaries in this directory.

Suggested filenames:

- `YYYY-MM-smart-contract-audit-report.pdf`
- `YYYY-MM-smart-contract-remediation-summary.md`
- `YYYY-MM-smart-contract-reaudit-addendum.pdf`

When a report is published:

1. Commit the report or a durable public link reference here.
2. Update `docs/audits/smart-contract-audit-status.yml`.
3. Set `significant_changes_since_last_audit: false` only after the audited code matches the deployed release candidate.
