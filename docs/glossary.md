# Glossary

This glossary defines the domain-specific terms used throughout the SorobanLoyalty platform.

## A-Z Terms

### Campaign
A merchant-created initiative that defines a reward amount and an expiration date. Users can claim rewards from active campaigns.

### Claim
The action a user takes to receive LYT rewards from an active campaign. This operation is recorded on-chain and prevents double-claiming.

### Freighter
A browser extension wallet used to securely sign and submit transactions to the Stellar network.

### Indexer
A backend service that polls the Stellar network for Soroban events (like campaign creation or reward claims) and updates the local PostgreSQL database to keep the frontend in sync.

### Ledger
The distributed, immutable record of all transactions and state changes on the Stellar network.

### LYT
The native fungible token of the SorobanLoyalty platform. It is used as a reward for users and can be redeemed for various benefits.

### Merchant
A business or entity that uses the platform to create and manage reward campaigns for their customers.

### Redeem
The action of spending or exchanging LYT tokens. Redeeming tokens typically involves burning them in exchange for real-world value or other platform benefits.

### Soroban
The smart contract platform built on the Stellar network, enabling programmable logic and decentralized applications.

### Stellar
The decentralized, open-source blockchain network that SorobanLoyalty uses for fast, low-cost asset transfers and smart contract execution.
